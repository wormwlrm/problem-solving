## 1495 기타리스트

<https://www.acmicpc.net/problem/1495>

## 내가 생각한 방법

<!-- ![이미지](./img.png) -->

- DFS를 이용한 백트래킹인 것 같다. 근데 아니었음 ㅋㅋ DP였다
- 하지만 DFS와 DP를 섞어서 해결해버림. 핵심은 중복 제거
- 만약 50에서 1, 1이 연속으로 나오게 되면...
  - 처음에는 49, 51로 상관이 없지만
  - 그 다음 단계에서 48, 50 그리고 50, 52가 나오게 됨
  - 따라서 50이 중복되는데 이걸 제거해주면 DFS로도 풀 수 있음
