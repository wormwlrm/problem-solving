## 17626 Four Squares

<https://www.acmicpc.net/problem/17626>

## 내가 생각한 방법

<!-- ![이미지](./img.png) -->

- 수학이라서 좀 어렵네
- DP로 풂
  - 초기 DP는 순서대로 적어놓음(1이 N개 있다는 의미)
  - 제곱수는 무조건 1로 만듦
  - dp 식은 현재 갱신중인 제곱수를 뺀 것의 dp에 1 더한것과 현재값 중 작은 것으로 갱신
- 예시
  - 만약 4라면 4는 2의 제곱수이므로 1
  - 5는 `dp[4]`에 `dp[1]` 더한 것과 같음
  - 6은 `dp[4]`에 `dp[2]` 더한 것과 같음
