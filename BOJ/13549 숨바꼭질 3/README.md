## 13549 숨바꼭질 3

<https://www.acmicpc.net/problem/13549>

## 내가 생각한 방법

<!-- ![이미지](./img.png) -->

- DFS + DP 였지만 순간이동이 까다로워서 살짝 시간이 걸리긴 했다. 한 40분?
  - 걸으면 1초가 소모되지만 순간이동은 노코스트 (사기스킬)
- 평소처럼 DFS를 돌리는데,
  - `children` 은 `텔레포트가 가능한 지역(N의 2제곱 배수)` + `current + 1` + `current - 1`이다.
  - 0에서 무한 루프 돌지 않게 예외처리 해주기
