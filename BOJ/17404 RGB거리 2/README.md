## 내가 생각한 방법

<!-- ![이미지](./img.png) -->

- DP 문제
  - DP는 이중 배열로 두는데,
    - 첫 번째 인덱스는 처음에 무엇을 선택했는지
    - 두 번째 인덱스는 RGB
  - 처음에 무엇을 선택했는지에 따라 초기 DP는 다음과 같이 설정됨
    - `[35, inf, inf]`
  - 그 후 작은 DP값 + 현재 RGB 값에 따라 값을 계속 갱신해나감
  - 마지막 줄까지 채워지면, 처음과 끝이 같은 걸 선택한 거 빼고 최소값 구하기
