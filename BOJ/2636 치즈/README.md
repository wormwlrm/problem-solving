## 2636 치즈

<https://www.acmicpc.net/problem/2636>

## 내가 생각한 방법

<!-- ![이미지](./img.png) -->

- 나름 신박한 시뮬레이션 문제였다. DFS로 충분히 해결 가능
- dfs 메소드는 공기와 맞닿은 치즈 좌표들을 리턴하게 만듦
  - 애초에 제일 바깥의 한 줄은 무조건 공기이기 때문에, `0, 0`에서 모든 공기층을 탐색함
  - 현재 좌표값이 1이라는 공기와 맞닿은 치즈라는 것,
  - 따라서 현재 값이 1일 때에는 방문할 노드를 따로 구하지 않음
- 기존의 드럽던 DFS 로직을 좀 개선해봄
