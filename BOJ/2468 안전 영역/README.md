## 2468 안전 영역

<https://www.acmicpc.net/problem/2468>

## 내가 생각한 방법

<!-- ![이미지](./img.png) -->

- BFS로 풀었음
  - 처음에 이분 탐색 혼합 유형인가 싶었는데
  - 꼭 그렇다고 보장할 수 없고 높이가 최대 100이기에 그냥 완전 탐색함
- 물에 잠기는 것을 기준으로 맵을 새로 만들어서, 물 높이 만큼을 뺀다
  - 만약 높이가 1보다 작으면 해당 영역은 탐색 불가능으로 설정
  - 그 후 방문 배열 초기화
  - 배열 모든 요소에 대해 BFS 실행
- 손으로 필기 안하고 머리로만 풀었다(감동)
