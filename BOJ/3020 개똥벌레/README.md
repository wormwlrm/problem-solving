## 내가 생각한 방법

<!-- ![이미지](./img.png) -->

- 누적 합 방법으로 풂
  - 높이 별 부딪히는 갯수가 정렬되어있기 때문에 이분탐색인 것 같긴 했는데, 이 방법은 도저히 풀이가 안 떠오름
  - 찾아보니 모든 높이에 대해 upper_bound, lower_bound를 계산해서 하기
- 전체 높이를 계산할 필요 없음
  - 높이 별 석순, 종유석 갯수를 배열로 구함
  - 석순의 경우 높이를 높은 곳에서부터 낮은 곳으로 누적합
  - 종유석의 경우 높이를 낮은 곳에서부터 높은 곳으로 누적합
