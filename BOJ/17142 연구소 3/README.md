## 17142 연구소 3

<https://www.acmicpc.net/problem/17142>

## 내가 생각한 방법

<!-- ![이미지](./img.png) -->

- BFS로 해보자
  - 고르는 것은 조합
- 예외처리가 상당히 까다롭다
  - 빈 벽이 없는 경우 (ex: `2 2 2 2 2`)
    - 나는 `empty_counter` 를 두고 애초에 빈 벽이 없으면 시도하지 않게 예외 처리
  - 바이러스 뚫고 가야하는 경우 (ex: `0 2 0 2 0`)
    - 해당 바이러스 칸이 유망한지를 판단(더 퍼져야 할 필요가 있는지, 아니면 끝내도 되는지)
